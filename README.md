# Portfólio Angular

Projeto desenvolvido utilizando Angular para a disciplina de Desenvolvimento Web II.

## Objetivo

Criar um portfólio pessoal para apresentar informações sobre mim, meus projetos e minhas habilidades na área de tecnologia.

## Funcionalidades Implementadas

Nesta etapa do projeto foram desenvolvidas as funcionalidades básicas do portfólio:

* Criação das páginas Início, Sobre, Projetos e Contato;
* Configuração do sistema de rotas utilizando Angular Router;
* Implementação de uma barra de navegação para acesso às páginas;
* Destaque visual da página ativa utilizando `routerLinkActive`;
* Utilização de componentes do Angular Material;
* Criação de cartões (`mat-card`) para exibição de conteúdo;
* Página Início com apresentação pessoal;
* Página Sobre com informações sobre minha trajetória acadêmica, interesses e objetivos profissionais;
* Estrutura inicial das páginas Projetos e Contato para futuras implementações.

## Tecnologias Utilizadas

* Angular
* TypeScript
* HTML
* CSS

Versões utilizadas:
